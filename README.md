# TP_OnePix

## Description

Sujet du TP OnePix : imagerie monopixel hyperspectral<br>
Version pour les étudiants de 2ème année Photonique<br>
Créé en septembre - octobre 2025<br>
A l'Enssat (Lannion) suite au stage d'Antonin Dodet chez POP (Photonics Open Project) encadré par Thomas Lebrat

## Contenu du repository

- Fichier source tex
- Fichier compilé pdf
- Dossier images

## Programmes Python

Ce sujet est lié à la version ONE_PIX_websocket_2 du logiciel embarqué sur le Raspberry.<br>
Il fait référence à des Jupyter Notebook hébergés sur ce même Github.
